# 🛒 عروضي — تطبيق الجوال

تطبيق أندرويد + iOS لمقارنة أسعار البقالة في الأسواق السعودية — بحث واحد، الأرخص أولاً.

**🔗 عرض حي (نسخة الويب):** https://abukootsh.github.io/oroudi/
> يعرض الواجهة كاملة؛ البحث الفعلي يحتاج خادم عروضي شغّالًا (راجع [oroudi-server](../oroudi-server)).

يقرأ التطبيق كل بياناته من **خادم عروضي** المستقل (`../oroudi-server`) —
المتاجر تُدار من لوحة التحكم، وأي متجر تضيفه هناك يظهر في التطبيق فورًا بدون تعديل كود.

## المزايا

- 🔍 **بحث** بالعربي أو الإنجليزي مع اقتراحات شائعة — النتائج مرتبة من الأرخص
- 🏪 **فلترة حسب المتجر**: الشرائح تُبنى ديناميكيًا من نتائجك، اختيار متعدد بألوان المتاجر
- 🏙️ **فلتر المدينة**
- ↕️ **ترتيب**: الأرخص / الأغلى / أعلى خصم
- 🏷️ **العروض**: المنتجات المخفّضة، وكل عرض يفتح صفحة المنتج في موقع المتجر (تبويب جديد)
- 📊 **أشهر المقارنات**: من عمليات بحث زوار تطبيقك — الكلمات بلا نتائج تُستبعد تلقائيًا
- 🫥 **تنبيه واضح** عند البحث عن كلمة بلا نتائج
- ♥ **المفضلة**: حفظ محلي دائم (AsyncStorage) — تبقى بعد إغلاق التطبيق
- 🔔 **تذكير تلقائي**: إشعارات محلية بعد ٣ و٧ أيام من آخر فتح + إشعارات جماعية من لوحة التحكم
- 🌐 **عربي / إنجليزي** بزر واحد، وخط IBM Plex Sans Arabic

## التشغيل

**١) شغّل الخادم أولًا:**
```bash
cd ../oroudi-server && npm start
```

**٢) شغّل التطبيق:**
```bash
npm install
npm start          # ثم امسح رمز QR بتطبيق Expo Go على جوالك
npm run web        # أو في المتصفح مباشرة
```

> الجوال والكمبيوتر على نفس شبكة الواي فاي — التطبيق يجد الخادم على شبكتك تلقائيًا أثناء التطوير.
> عند النشر ضع رابط خادمك في `PROD_SERVER` داخل `src/api.ts`.

## بناء نسخ للنشر (APK / App Store)

عبر [EAS Build](https://docs.expo.dev/build/introduction/) — يبني في السحابة بدون Xcode أو Android Studio:

```bash
npm install -g eas-cli
eas login          # حساب Expo مجاني
eas build:configure
eas build -p android --profile preview   # ملف APK للتجربة المباشرة
eas build -p android                     # AAB لمتجر Google Play
eas build -p ios                         # يتطلب حساب Apple Developer ($99/سنة)
```

## بنية المشروع

```
App.tsx                     نقطة الدخول: الهيدر، تبديل اللغة، التبويبات
src/
  api.ts                    عميل خادم عروضي (بحث، عروض، مقارنات، تسجيل جهاز)
  i18n.ts                   النصوص عربي/إنجليزي
  theme.ts                  ألوان الهوية وألوان المتاجر والخطوط
  favorites.tsx             حفظ المفضلة (Context + AsyncStorage)
  notifications.ts          إشعارات التذكير + تسجيل الجهاز للإشعارات الجماعية
  components/
    ProductCard.tsx         بطاقة المنتج (صورة، شارات، سعر، خصم، قلب، فتح في المتجر)
    Chip.tsx                زر الفلترة/الاقتراحات
    PopularPills.tsx        أشهر المقارنات
  screens/
    SearchScreen.tsx        البحث والفلترة والترتيب
    DealsScreen.tsx         أفضل العروض
    FavoritesScreen.tsx     المفضلة
```
